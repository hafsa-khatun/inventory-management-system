# Inventory Management System

## 📌 Project Overview
Inventory Management System is a full-stack web application developed to manage products, categories, suppliers, customers, purchases, and sales efficiently.

## 🚀 Features
- User Management
- Category Management
- Product Management
- Supplier Management
- Customer Management
- Purchase Management
- Sales Management
- Inventory Tracking
- REST API

## 🛠️ Technologies Used

### Backend
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

### Frontend
- Angular
- TypeScript
- HTML
- CSS
- Bootstrap

## 🗄️ Database
- MySQL

## 📂 Project Structure

```
Inventory-Managements/
│
├── inventory-api/      # Spring Boot Backend
└── Inventory-ui/       # Angular Frontend
```

## ⚙️ Installation

### Backend

```bash
cd inventory-api
mvn spring-boot:run
```

### Frontend

```bash
cd Inventory-ui
npm install
ng serve
```

Open your browser and visit:

```
http://localhost:4200
```

## API

Backend runs on:

```
http://localhost:8080
```
